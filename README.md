### Typecho-WeiboFile新浪微博图片上传插件
---

将 Typecho 的附件上传至新浪微博云存储中，无需申请appid，不占用服务器大小，可永久保存，只需一个不会登录的微博小号即可。

#### 使用方法：
第一步：下载本插件，放在 `usr/plugins/` 目录中；
第二步：激活插件；
第三步：填写微博小号等等配置；
第四步：完成。

#### 使用注意：
此插件V1.0.1版本使用php5.6编写，php7.0“可能”会报Sinaupload.php中的语法错误，建议使用php5.6，因为7.0实在太高了=_=!
